# AndroidMovil — Guías y Herramientas Avanzadas 📱⚡

¡Bienvenido al repositorio oficial de la comunidad **AndroidMovil**! Este proyecto es una landing page diseñada para centralizar guías, herramientas y soporte especializado para usuarios que buscan llevar su dispositivo Android al siguiente nivel, con un enfoque especial en métodos **sin necesidad de PC**.

🌐 **Visita el sitio web oficial:** [https://michaelarc-ni.github.io/AndroidMovil/](https://michaelarc-ni.github.io/AndroidMovil/)

## ✨ Características del Sitio
- **Diseño Futurista:** Interfaz oscura con acentos neón y tipografía técnica.
- **Modo Matrix:** Incluye un efecto visual interactivo inspirado en la "lluvia digital" de Matrix mediante Canvas API.
- **Terminal Interactiva:** Simulación de logs de comandos en tiempo real para una estética hacker/dev.
- **Responsive Design:** Totalmente optimizado para dispositivos móviles y tablets.
- **Acceso Directo:** Enlaces rápidos a tutoriales de Root, Bootloader, ROMs y recuperación de IMEI.

## 🛠️ Tecnologías Utilizadas
- **HTML5 & CSS3:** Variables de CSS, Grillas (Grid) y Flexbox para el diseño.
- **JavaScript Vanila:** Lógica para el Modo Matrix, la terminal de comandos y el cambio de temas.
- **Google Fonts:** Uso de *Oxanium* para un estilo tech y *Source Serif 4* para legibilidad.

## 📚 Contenido Destacado en la Web
El sitio sirve como hub para tutoriales críticos como:
- Desbloqueo de Bootloader en marcas como Tecno e Infinix sin PC.
- Root moderno con Magisk (parcheo de `init_boot`).
- Solución al error "dm-verity corruption".
- Backup de IMEI y particiones EFS desde Termux.
